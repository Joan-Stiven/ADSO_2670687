Consepto de GIT

Git es una herramienta para controlar versiones de código. Te permite guardar cambios, trabajar en equipo y crear ramas separadas para experimentar sin dañar el proyecto principal. Es muy útil para desarrolladores porque mantiene un historial de cambios y permite colaborar de manera organizada en proyectos de software.

PRICIPALES COMANDOS DE GIT

  • GIT INIT: Sirve para inicializar un nuevo repositorio en Git en un dirrectorio.
  • GIT ADD: Se utiliza para agregar cambios en archivoz al área de preparacion.
  • GIT CONFIG: Se utiliza para configurar la informacion de Git a nivel de usuario o a nivel de repositorio.
  • GIT COMMIT: Permite realizar un comentario sobre los cambios realizados.
  • GIT LOG: Permite explorar el historial de repositorio.
  • GIT REMOTE: Permite crear, ver y eliminar conexiones con otros repositorios.
  • GIT STATUS: Muestra el estado de dirrectorio de trabajo y del area de entorno de ensayo.
  • GIT BRANCH: Te permite crear, enumerar y eliminar ramas, asi como cambiar su nombre.
  • GIT CHECKOUT: Permite desplazarte entre las ramas creadas.
  • GIT PUSH: Guarda todo en la nube.
